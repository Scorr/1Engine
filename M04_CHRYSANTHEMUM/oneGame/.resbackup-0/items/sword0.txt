// == Information ==
// The parts of a sword are as follows.
//	part0/blade
//	part1/hilt/handle
//	part2 <- optional
//	part3 <- optional
// == Type Names ==
// "blade" is understood to mean "part0" ; "hilt" and "handle" are understood to mean "part1"
// Type names are interchangable as long as the parser converts them to the same meaning.
